# collab.blackroad.io

> Cloudflare Worker serving [collab.blackroad.io](https://collab.blackroad.io)

## Overview

This is the Cloudflare Worker powering the **collab** subdomain of blackroad.io.

## Deployment

```bash
# Deploy worker
wrangler deploy

# Tail live logs
wrangler tail
```

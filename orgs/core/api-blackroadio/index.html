BlackRoad API

Tokenless gateway · all providers behind one interface · base: https://api.blackroad.io/v1

Agents

GET/agents
List all 30,000 fleet agents and named agents with capabilities
GET/agents/:id
Get single agent profile, skills, status
POST/agents/task
Route a task to the best-matched agent based on capabilities
GET/fleet
Pi fleet node status, capacity, active workloads

Tools

GET/tools
All 162 br CLI tools with categories and usage
POST/tools/exec
Execute a br tool remotely via Pi fleet

Gateway (AI)

POST/chat
Multi-provider AI chat (Anthropic · OpenAI · Ollama · Groq · Gemini · Mistral)
GET/models
All available models across all providers

System

GET/health
System health — all services, Pi nodes, Cloudflare workers
GET/metrics
Real-time metrics: requests/sec, agent load, queue depth

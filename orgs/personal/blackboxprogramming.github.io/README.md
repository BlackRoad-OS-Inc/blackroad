# blackboxprogramming.github.io

> GitHub Pages site for the blackboxprogramming organization

## Overview

This is the GitHub Pages site hosted at [blackboxprogramming.github.io](https://blackboxprogramming.github.io), serving as the public-facing portal for the `blackboxprogramming` GitHub account.

## Structure

```
blackboxprogramming.github.io/
├── index.html        # Main landing page
├── assets/           # Static assets (images, CSS, JS)
├── auth/             # Authentication pages
├── dashboard/        # Dashboard interface
├── pages/            # Additional pages
└── templates/        # Page templates
```

## Deployment

Automatically deployed via GitHub Pages on push to `main`.

```bash
# Local preview
python3 -m http.server 8080

# Or with Node.js
npx serve .
```

# BlackBox ActivePieces

> Open source no-code automation platform (Fork)

## Quick Reference

| Property | Value |
|----------|-------|
| **Upstream** | activepieces/activepieces |
| **Type** | Automation Platform |
| **Stack** | TypeScript + Angular |
| **License** | MIT |

## Tech Stack

```
Frontend: Angular + TypeScript
Backend: Node.js + NestJS
Database: PostgreSQL
Queue: Redis
```

## Commands

```bash
pnpm install        # Install dependencies
pnpm start          # Start development
pnpm build          # Production build
pnpm test           # Run tests
```

## BlackRoad Integration

Sovereign fork for:
- Custom workflow integrations
- BlackRoad agent triggers
- Internal automation

## Key Components

- **Pieces**: Connectors to external services
- **Flows**: Visual workflow definitions
- **Triggers**: Event-based execution
- **Actions**: Workflow steps

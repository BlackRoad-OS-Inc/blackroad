# BlackRoad Slack Bot

Slack bot integrations for BlackRoad OS.

## Features

- `/blackroad` - Check infrastructure status
- `@BlackRoad` mentions - Get help and info
- Deployment notifications
- Alert forwarding

## Setup

1. Create Slack app at api.slack.com
2. Set environment variables:
   - `SLACK_BOT_TOKEN`
   - `SLACK_SIGNING_SECRET`
   - `SLACK_APP_TOKEN`
3. Run `npm install && npm start`

# BlackRoad OS VSCode Extension

AI-powered development tools for BlackRoad infrastructure.

## Features

- `BlackRoad: Show Status` - Check infrastructure status
- `BlackRoad: Deploy` - Deploy to Cloudflare/Railway/Vercel
- `BlackRoad: List Agents` - View active AI agents

## Installation

1. Download from VS Code Marketplace
2. Or: `code --install-extension blackroad-os.blackroad-os`

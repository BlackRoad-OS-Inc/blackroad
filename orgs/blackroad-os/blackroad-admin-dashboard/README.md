# BlackRoad Unified Admin Dashboard

**Enterprise-grade admin dashboard for managing all BlackRoad products**

## Overview

Centralized control panel for all 11 BlackRoad Enterprise products:

- **vLLM** - AI model serving
- **Ollama** - Local LLM runtime
- **LocalAI** - AI inference
- **Headscale** - Network control
- **MinIO** - Object storage
- **EspoCRM** - Customer management
- **Authelia** - Authentication
- **Whisper** - Speech-to-text
- **Restic** - Backup solution
- **NetBird** - Network mesh
- **Focalboard** - Project management

# BlackRoad minio Enterprise API

## Overview

Proprietary API layer for BlackRoad minio Enterprise.

## Features

- **Authentication**: JWT, OAuth 2.0, SSO
- **Rate Limiting**: Tier-based limits
- **Analytics**: Usage tracking and billing
- **Webhooks**: Event-driven integrations

## Endpoints

```
POST /api/auth/login
GET  /api/minio/*
POST /api/minio/*
```

## Integration

Connects to upstream minio via standard APIs.

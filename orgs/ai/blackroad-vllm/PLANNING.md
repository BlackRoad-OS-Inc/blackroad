## Performance Optimization

### Batching Strategy

| Batch Size | Throughput | Latency (p99) | GPU Util |
|------------|------------|---------------|----------|
| 1 | 50 req/s | 100ms | 30% |
| 8 | 200 req/s | 200ms | 70% |
| 32 | 500 req/s | 400ms | 90% |
| 64 | 700 req/s | 600ms | 95% |

**Target**: Dynamic batching with 32-64 batch size

### Memory Optimization

```
PagedAttention Settings:
- Block size: 16 tokens
- GPU memory utilization: 90%
- Swap space: 4GB
- Max model len: 32768 tokens
```

### Caching Strategy

1. **Prompt Cache**: Cache common prompt prefixes
2. **KV Cache**: Cache attention KV for repeated queries
3. **Response Cache**: Cache identical requests (TTL: 5min)

Architecture

The Tokenless Gateway

Agents never see provider API keys. All traffic routes through the gateway at localhost:8787. One trust boundary. No leaks.

● Agents authenticated by capability registry — no raw tokens
● Gateway owns all secrets — stored in encrypted vault
● Gateway binds to 127.0.0.1 by default — no external exposure

# CLAUDE.md — blackroad-operator

The `br` CLI for BlackRoad OS. Talks to the gateway, manages agents, deploys services.

## Stack
- Node.js 22 / TypeScript 5
- Commander (CLI framework)
- Chalk (colors) / Ora (spinners) / Conf (config storage)
- Vitest (testing)

## Commands
`npm run dev` — watch mode, `npm run typecheck` — check types, `npm test` — run tests

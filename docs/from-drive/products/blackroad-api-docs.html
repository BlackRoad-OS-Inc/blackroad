Getting Started

Welcome to the BlackRoad API! All API requests require authentication via API key.

Base URL

https://api.blackroad.io/v1

Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Example Request

curl -X POST https://api.blackroad.io/v1/analytics/track \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
"event": "page_view",
"properties": {
"page": "/home",
"referrer": "google.com"
}
}'

Response Format

{
"success": true,
"data": { ... },
"timestamp": "2026-01-09T23:00:00Z"
}

# BlackRoad Dashboards

> Collection of monitoring and analytics dashboards

## Quick Reference

| Property | Value |
|----------|-------|
| **Type** | Dashboard Collection |
| **Count** | 100+ dashboards |
| **Purpose** | Monitoring |

## Dashboard Types

- **Infrastructure**: Server and service health
- **Agent Metrics**: 30,000 agent monitoring
- **Analytics**: Usage and performance
- **Security**: Threat detection
